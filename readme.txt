== Description ==

Add a conveyancing quote calculator to your website allowing your visitors to get instant fee estimates on screen and have their conveyancing quote automatically emailed to them as well. You will be alerted via email to any quotes which have been sent to prospective clients and all the clients information will be stored in Hoowla's online conveyancing platform.

Are you a Conveyancer, Solicitor, Estate Agent, Mortgage Broker or simply doing some web work for them and want to find out more about our Conveyancing Platform? Get in touch (hello@hoowla.com) and we'd be happy to answer any of your questions or help you setup the conveyancing calculator.

*This plugin requires a Hoowla account, it's free to [signup](https://www.hoowla.com/) and requires a [quote form to be setup](http://blog.hoowla.com/setup-conveyancing-quote-calculator-hoowla/) within the system (it's also free).*

Requires Wordpress 3.4+ and PHP5+.

== Installation ==

1. Upload the un-zipped 'hoowla-quote-plugin' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create a Quote Form on Hoowla - http://blog.hoowla.com/setup-conveyancing-quote-calculator-hoowla/
4. Insert a button into any page/post in Wordpress by clicking the green Hoowla log in the editor.

== Frequently Asked Questions ==

= Does this plugin work with newest WP version and also older versions? =
Yes, this plugin works fine with WordPress 3.9!
It also works great with WP 3.4 - and perhaps below - but we've only tested extensively with WP 3.4 and above. So you always should run the latest WordPress version.

= Does this plugin depend on any extenal sources? =
Yes, the plugin will call a JavaScript file from Hoowla's content delivery network which will load the plugin. When submitted the form will also submit to the Hoowla Servers.
